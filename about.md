# Sweppa Kiddies Montessori — Website Project

## Project Overview

This is the static website for **Sweppa Kiddies Montessori**, a Montessori school in Malawi. The site serves as both a public marketing platform and a parent communication hub.

Built as a GitHub Pages-compatible static site with modern HTML5, CSS3, and vanilla JavaScript. No build tools required.

## Image Placement

Download images using the search terms provided and place them in the `images/` folder with the exact filenames listed above.

## Hosting on GitHub Pages

1. Create a new repository on GitHub (e.g., `sweppa-kiddies-montessori`)
2. Push all files to the `main` branch
3. Go to Settings → Pages → Source: Deploy from branch → `main` / `/ (root)`
4. Site will be live at `https://<username>.github.io/sweppa-kiddies-montessori/`
